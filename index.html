Curso HTML5 e CSS3

O que é HTML5

HTML5 é : (abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web

!DOCTYPE html

O tipo do documento é informado com a declaração ! DOCTYPE> refere-se a um DTD, porque HTML 4.01 foi baseada em SGML. O DTD especifica as regras para a linguagem de marcação, para que os navegadores processar o conteúdo corretamente

O que é lang

o atributo lang independente do seu valor, pt ou pt-BR não muda em nada, a ideia do uso dele é normatizar e indicar os tipos de dados pra quem tiver interesse, se não houver esse atributo plugins, google tradutor ou buscadores tem mais dificuldade de detectar o idioma e região da página

Head

O que é head O head de um documento HTML é a parte que não é exibida no navegador da Web quando a página é carregada -->

meta charset="utf-8">

O que é meta O charset UTF-8 é o que usamos na web atual e faz parte de um padrão chamado Unicode. A ideia é que com ele possamos representar qualquer caractere de qualquer idioma.

title

O que é title (Elemento HTML Título) define o título do documento, mostrado na barra de título de um navegador ou na aba da página.

body

O que é body: O elemento body do HTML representa o conteúdo de um documento HTML. è permitido apenas um body por documento.
